============================================
  AKCES HUB - INSTALACJA NA WINDOWS
============================================

KROK 1: Rozpakuj caly zip do dowolnego folderu
        (np. C:\AKCES_HUB\)

KROK 2: Wejdz do folderu "installer\"

KROK 3: Kliknij dwa razy "INSTALL.bat"
        Poczekaj az skonczy (2-3 min).
        Przegladarka otworzy sie automatycznie.

KROK 4: W przegladarce stwórz konto administratora
        (formularz "Pierwsza konfiguracja").

KROK 5: Wklej plik licencji (.lic) ktory dostales
        od dostawcy w zakladce "Ustawienia".


============================================
  DZIAŁANIE
============================================

- AKCES HUB uruchamia sie automatycznie przy
  starcie Windowsa.
- Otworzysz aplikacje skrótem na pulpicie
  ("AKCES HUB") lub klikajac START.bat.
- Zatrzymanie: kliknij STOP.bat lub Task Manager.

Aplikacja dziala na: http://localhost:5000


============================================
  AKTUALIZACJE
============================================

Aktualizacje pobierasz BEZ konsoli:
  1. Wejdz w aplikacji do "Narzedzia"
  2. Kliknij "Update systemu"
  3. Potwierdz
  4. Poczekaj ~1 min — aplikacja zrestartuje sie
     automatycznie.


============================================
  WSPARCIE
============================================

W razie problemow:
  - Sprawdz logi: folder logs\ w glownym katalogu
  - Backupy bazy: folder backups\
  - Restart: STOP.bat -> START.bat

Kontakt: zgodnie z umowa zakupu.
